-----------------------------------
HOW TO USE
-----------------------------------

1. Copy a token address from Token 1
2. Paste it into the "First Token Contract Address" field
3. Copy a token address from Token 2
4. Paste it into the "Second Token Contract Address" field
5. Click "Show Rate" button
6. View the live exchange rate and chart!

The chart will update every 5 seconds automatically.

-----------------------------------
TROUBLESHOOTING
-----------------------------------

If you see an error:
- Make sure the address is exactly 42 characters (0x + 40 hex digits)
- Check that the token is actively traded on DEX platforms
- Verify you have internet connection
- Try a different token pair
- Check browser console (F12) for detailed error messages
